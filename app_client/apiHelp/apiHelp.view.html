Resources

Assets

Resource Description
GET /api/assets List all existing assets
GET /api/assets/:asset_id Show the asset with that id
POST /api/assets Create a new asset
PUT /api/assets/:asset_id Update the asset with that id
DELETE /api/assets/:asset_id Delete the asset with that id

SKUs

Resource Description Example
GET /api/skus List all existing SKUs curl http://<larrs_server>:3000/api/skus
GET /api/skus/:sku_id Show the SKU with that id curl http://<larrs_server>:3000/api/skus/<sku_id>
POST /api/skus Create a new SKU curl -H "Authorization: Bearer <auth_token>" -H "Content-Type: application/json" -X POST http://<larrs_server>:3000/api/skus -d '{ "name": "<sku_name>", "notes": "<sku_notes>" }'
PUT /api/skus/:sku_id Update the SKU with that id curl -H "Authorization: Bearer <auth_token>" -H "Content-Type: application/json" -X PUT http://<larrs_server>:3000/api/skus/<sku_id> -d '{ "name": "<mod_sku_name>", "notes": "<mod_sku_notes>" }'
DELETE /api/skus/:sku_id Delete the SKU with that id curl -H "Authorization: Bearer <auth_token>" -X DELETE http://<larrs_server>:3000/api/skus/<sku_id>
