Resources

Assets

Resource Description
GET /api/assets List all existing assets
GET /api/assets/:asset_id Show the asset with that id
POST /api/assets Create a new asset
PUT /api/assets/:asset_id Update the asset with that id
DELETE /api/assets/:asset_id Delete the asset with that id

Manufacturers

Resource Description Example
GET /api/manufacturers List all existing manufacturers curl http://<larrs_server>:3000/api/manufacturers
GET /api/manufacturers/:mfg_id Show the manufacturer with that id curl http://<larrs_server>:3000/api/manufacturers/<mfg_id>
POST /api/manufacturers Create a new manufacturer curl -H "Authorization: Bearer <auth_token>" -H "Content-Type: application/json" -X POST http://<larrs_server>:3000/api/manufacturers -d '{ "name": "<mfg_name>", "notes": "<mfg_notes>" }'
PUT /api/manufacturer/:mfg_id Update the manufacturer with that id curl -H "Authorization: Bearer <auth_token>" -H "Content-Type: application/json" -X PUT http://<larrs_server>:3000/api/manufacturers/<mfg_id> -d '{ "name": "<mod_mfg_name>", "notes": "<mod_mfg_notes>" }'
DELETE /api/manufacturers/:mfg_id Delete the manufacturer with that id curl -H "Authorization: Bearer <auth_token>" -X DELETE http://<larrs_server>:3000/api/manufacturer/<mfg_id>
